# my-linux-journey
My personal Linux journey from a cybersecurity perspective


**Ubuntu**
Easy ASF, User Friendly ASF, Clean, Smooth, Perfect in Development and Daily-use, feels polished, not Cybersecurity-focused by default but can handle it

**Debian**
RAW.. but too much, Minimal ASF.. Stable ASF.. Powerful ASF, gives you full control, not comfy, not fancy, but when it comes to Cybersecurity the productivity feels terrible because of the annoying manual setup and workflow friction

**Kali Linux**
Too much tools, Heavy kinda, Offensive Cybersecurity monster, feels more like a hacking toolkit than a normal OS, not the best Daily-use distro IMO

**Parrot Home**
Light, Smooth, can be an amazing Daily-use OS, privacy vibes, minimal but flexible, can evolve into a devil workstation for Cybersecurity

**Parrot Security**
Like Kali but actually feels like a REAL OS, Light, balanced, good for both Red Teaming and Blue Teaming, not overwhelming like Kali
